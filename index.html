What Is βeautifier ?

βeautifier organizes and beauties your CSS - javascript Code and Soon More To come - in a manner that makes it easier to read and define page areas and see how they relate to one another.

Per exapmle, it turns a code looking like this:

.hvr-grow{display:inline-block;vertical-align:middle;-webkit-transform:translateZ(0);
transform:translateZ(0);box-shadow:0 0 1px rgba(0,0,0,0);-webkit-backface-visibility:hidden;
backface-visibility:hidden;-moz-osx-font-smoothing:grayscale;-webkit-transition-duration:.3s;transition-duration:.3s;

into something like This:

.hvr-grow {
    display: inline-block;
    vertical-align: middle;
    -webkit-transform: translateZ(0);
    transform: translateZ(0);
    box-shadow: 0 0 1px rgba(0, 0, 0, 0);
    -webkit-backface-visibility: hidden;
    backface-visibility: hidden;
    -moz-osx-font-smoothing: grayscale;
    -webkit-transition-duration: .3s;
    transition-duration: .3s;
}
